# **Motivational Website — README**

## **Overview**
This project is a simple motivational webpage designed to practice core CSS concepts while building a clean, centered layout inspired by a classic motivational poster. The goal was to combine typography, layout techniques, and image styling to create a visually striking and encouraging message.

## **What I Studied**
During this project, I explored several foundational CSS topics, including:

- **CSS Colors:** Applying text and background colors to create contrast and visual hierarchy.  
- **Font Properties:** Using Google Fonts, adjusting font size, and applying text transformations.  
- **Font Weight:** Understanding how different weights affect emphasis and readability.  
- **CSS Box Model:** Working with margins, padding, borders, and element sizing to control layout.  
- **Text Alignment:** Centering content and ensuring consistent visual flow across the page.  

These concepts helped me understand how CSS shapes the look and feel of a webpage.

## **What I Built**
- A motivational webpage featuring a centered image, a bold headline, and an inspirational quote.  
- A layout structured inside a main container to keep the content visually balanced.  
- A black‑and‑white theme for strong contrast and a clean, modern aesthetic.  
- A custom Google Font (Libre Baskerville) to give the headline a classic, elegant feel.  
- An image styled with a white border and scaled to fit the container smoothly.  

## **Skills Practiced**
- Linking external CSS files  
- Importing and using Google Fonts  
- Applying the CSS box model to control spacing  
- Centering elements using width and margin techniques  
- Styling text with transformations and font properties  
- Creating a cohesive visual theme
