# ** Website Foundations & Movie Ranking Project**

## 🌐 Website Foundations  
This section contains my foundational HTML lessons, where I practiced core concepts such as headings, paragraphs, line breaks, horizontal rules, and document structure. Each lesson builds toward understanding how HTML content is organized and displayed in the browser.

---

## 📖 **Lesson 1 — HTML Structure Practice**

This lesson focuses on understanding the hierarchy of HTML headings and how they create structure within a document.

This exercise helped reinforce:

- Semantic structure  
- Proper heading nesting  
- Readability and document flow  

---

## 📝 **Lesson 2 — Paragraphs & Text Formatting**

This lesson focused on:

- Writing multiple paragraphs  
- Understanding spacing  
- Practicing long‑form text formatting  
- Using `<p>` tags correctly  

I used placeholder text to simulate real content and practiced structuring long paragraphs for readability.

---

## 🖋️ **Lesson 3 — Addresses, Line Breaks & Horizontal Rules**

This lesson introduced:

- `<br>` for line breaks  
- `<hr>` for thematic separation  
- Structuring address blocks  
- Combining text with formatting elements  

Example content included a formatted address and a biographical section about William Blake.
