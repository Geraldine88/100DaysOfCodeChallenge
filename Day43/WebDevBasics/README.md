# **Spanish Colors Website — README**

## **Overview**
This project is a simple, beginner‑friendly webpage designed to help users learn basic color vocabulary in Spanish. It was built using HTML for structure and an external CSS stylesheet for all styling. The exercise focused on understanding how CSS works, how it interacts with HTML, and how to apply different types of selectors to style specific elements on a page.

## **What I Learned**
- **What CSS is and why it matters:** I learned that CSS controls the visual presentation of a webpage, separating style from structure.  
- **Why it’s called “Cascading” Style Sheets:** Styles flow downward like a waterfall, meaning the browser decides which rules apply based on specificity and order.  
- **Types of CSS:** I explored inline, internal, and external CSS, and practiced linking an external stylesheet to an HTML file.  
- **CSS Selectors:** I learned how to target elements using classes, IDs, universal selectors, and attribute selectors.  
- **Styling Text and Images:** I practiced changing text color, adjusting font weight, and resizing images using CSS properties.  
- **Real‑world application:** I used CSS IDs to match Spanish color names with their correct colors, reinforcing both vocabulary and selector usage.

## **What I Built**
- A small educational webpage that displays five Spanish color names.  
- Each color name is styled to match its meaning (e.g., *Rojo* appears in red).  
- All color titles share consistent styling through a class selector.  
- All images on the page were resized to a uniform 200×200 pixels using CSS.  
- The entire page uses an external stylesheet, keeping the HTML clean and organized.

## **Skills Practiced**
- Linking external CSS files  
- Using class and ID selectors  
- Applying color, font, and layout properties  
- Understanding the cascade and specificity  
- Working with images in CSS
